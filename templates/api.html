Intro

This is a simple API generated using Python and PostgreSQL for you to play around.

You can pass in the book isbn number in the request URL and get the book's information(title/author/year/review...) back.

Request URL : https://book-fun.herokuapp.com/api/<isbn>

Request Method : GET

Response format : JSON

Example

Example GET request using fetch() promise: 

fetch('https://book-fun.herokuapp.com/api/1595543414')
  .then(response => response.json())
  .then(json => console.log(json))

Your API call was successful!

*You can also simply put the request URL into your broswer and see the results there
